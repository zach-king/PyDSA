PyDSA: Python Data Structures & Algorithms

Simple data structures and algorithms implemented in Python 3.

These implementations are based off of those written by Ken Lambert in his book, "Fundamental of Python: Data Structures." Most of them have been modified to an extent, and a couple may not be completely functional. There are also "interfaces" implemented to better utilize the heirarchial approach of object-orientation.

Help with Individual Data Structures or Algorithms

To get help with individual data structures, their methods, complexity analysis, etc. please use the docstrings. An example would be as follows:

import ArrayBag

print(help(ArrayBag))

or

import ArrayBag

print(ArrayBag.__doc__)
